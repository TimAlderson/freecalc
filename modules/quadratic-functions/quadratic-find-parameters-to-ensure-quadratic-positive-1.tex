Find the values of the parameter $k$ for which $
\alertNoH{3}{x^2} +\alertNoH{10}{(k+1)}x+\alertNoH{12}{2k}>0
$ holds for all real $x$.
\begin{itemize}
\item<2-> In order for the quadratic to be positive, its graph must lie entirely above the $x$ axis.
\end{itemize}
\end{columns}
\vskip -0.05cm
\begin{itemize}
\item<3-> \alertNoH{3}{Leading coefficient is positive} \uncover<4->{$\Rightarrow$ graph opens up} \uncover<5->{$\Rightarrow$ is above $x$ axis if it does not intersect it} \uncover<6->{$\Rightarrow$ \alertNoH{9}{the quadratic has no real solutions}.} 
\item<7-> The roots of a quadratic are $\alertNoH{7,8}{x_1, x_2=} \fcAnswer{8}{\frac{-b\pm\alertNoH{9}{\sqrt{{\alertNoH{10}{b}}^2-4{\alertNoH{11}{a}}{\alertNoH{12}{c}}}}}{2a}}$\uncover<8->{.}
\end{itemize}
